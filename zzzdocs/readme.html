RAGE Key-Based Encryption and Decryption

This guide explains how to use rage for encrypting and decrypting files using a key file, rather than a passphrase. The key file is generated with rage-keygen.exe, and the encryption and decryption operations are performed using rage.exe.

Prerequisites

Ensure you have rage.exe and rage-keygen.exe available in the same directory where you're working.

Step 1: Generate a Key

To generate a key file (key.key), run the following command in your terminal or command prompt:

rage-keygen.exe -o key.key

This will create a key file called key.key in the current directory. This key will be used for both encryption and decryption.

Step 2: Encrypt a File

To encrypt a file (a.txt), use the key file (key.key) and output the encrypted content to another file (b.txt):

rage.exe --encrypt -i key.key a.txt -o b.txt
  • a.txt: The original plaintext file you want to encrypt.
  • key.key: The key file used for encryption.
  • b.txt: The encrypted file output.

Step 3: Decrypt a File

To decrypt the previously encrypted file (b.txt) back into plaintext (c.txt), run the following command:

rage.exe --decrypt -i key.key b.txt -o c.txt
  • b.txt: The encrypted file.
  • key.key: The key file used for decryption.
  • c.txt: The output file that contains the decrypted plaintext.

Example Workflow

  1. Generate a key:
    2. rage-keygen.exe -o key.key
  2. Encrypt a.txt to b.txt:
    3. rage.exe --encrypt -i key.key a.txt -o b.txt
  3. Decrypt b.txt back to c.txt:
    4. rage.exe --decrypt -i key.key b.txt -o c.txt

In this example:

  • a.txt is your original file.
  • b.txt is the encrypted version of a.txt.
  • c.txt is the decrypted version of b.txt, which should match a.txt.

Encrypting and Decrypting with a Passphrase (Password Mode)

If you prefer to encrypt and decrypt your files using a passphrase (password) instead of a key file, follow the steps below:

Step 1: Encrypt a File with a Passphrase

To encrypt a file using a passphrase, run the following command:

rage -p -o b.txt a.txt
  • a.txt: The original file you want to encrypt.
  • b.txt: The output encrypted file.

After running this command, you will be prompted to enter a passphrase. This passphrase will be required to decrypt the file later.

Step 2: Decrypt a File with a Passphrase

To decrypt a file that was encrypted with a passphrase, use the following command:

rage -d -o c.txt b.txt
  • b.txt: The encrypted file you want to decrypt.
  • c.txt: The output decrypted file.

You will be prompted to enter the same passphrase you used during encryption. Once you enter the correct passphrase, the file will be decrypted.

Important Note

Make sure to keep your passphrase safe and remember it, as you won't be able to decrypt the file without it!
